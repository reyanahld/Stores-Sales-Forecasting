Dalam penelitian ini akan dilakukan peramalan untuk variabel penjualan bulanan menggunakan tools R Studio dan dataset yang digunakan diambil dari Kaggle. Data yang digunakan dalam penelitian ini berasal dari Kaggle dengan nama dataset : Store Sales Forecasting Dataset. Data terdiri dari 21 variabel yang terdiri dari variabel Row ID, Order ID, Order Date, Ship Date, Ship Mode, Customer ID, Customer Name, Segment, Country, City, State, Postal Code, Region, Product ID, Category, Sub Category, Product Name, Sales, Quantity, Discount, dan Profit.

Metode yang digunakan dalam peramalan ini ialah ARIMA atau Autoregressive Integrated Moving Average. ARIMA merupakan salah satu metode peramalan untuk data time series.

Dataset dapat diakses pada link berikut: https://www.kaggle.com/datasets/tanayatipre/store-sales-forecasting-dataset
